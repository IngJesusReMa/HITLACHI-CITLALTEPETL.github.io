SENDERISMO DE MEDIA MONTAÑA

An API (Application Programming Interface) provides an interface for various programming languages that allows developers and programmers to integrate the data directly into their systems in the desired format in just a few steps.

The Meteomatics weather API contains data on weather and climate. It provides access to over 25 weather models (e.g. GFS, ECMWF, DWD, MeteoFrance, UKMetoffice and many more) and more than 1800 weather parameters. Our interface is thus considered the gateway to the largest database of historical weather data, forecast data and climate data worldwide.
